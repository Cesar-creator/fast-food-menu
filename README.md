## 🎯 El Problema
Los restaurantes familiares y pequeños negocios a menudo se enfrentan a un gasto constante y poco ecológico: **la reimpresión de cartas físicas**. Cada vez que hay una fluctuación en los precios de los ingredientes, añadir un nuevo platillo o simplemente corregir un error, el negocio se ve forzado a desechar sus menús antiguos y gastar capital en nuevas impresiones.

## 💡 La Solución
**Fast Food Digital Menu** es una aplicación web interactiva desarrollada para reemplazar las tradicionales cartas de papel por un menú digital atractivo, fácil de navegar y altamente responsivo.

### Beneficios Clave:
- 💰 **Ahorro de costos:** Cero gastos en reimpresión de menús físicos.
- 🚀 **Actualizaciones instantáneas:** Modificar precios, nombres o agregar productos se refleja instantáneamente sin interrumpir el servicio.
- 📱 **Mobile First:** Diseño pensado prioritariamente para los dispositivos móviles de los clientes, asegurando la mejor experiencia de usuario desde cualquier smartphone.
- ♻️ **Sostenibilidad:** Una opción amigable con el medio ambiente al reducir drásticamente el uso de papel y plásticos.

---

## ✨ Características Principales

- **Navegación Intuitiva:** Categorización de productos (Hamburguesas, Perros Calientes, Picadas, etc.) con anclaje rápido y observadores de intersección (Intersection Observer).
- **Diseño Dinámico:** Interfaz moderna y fluida con animaciones que mejoran la retención e interacción del usuario.
- **Gestión de Imágenes:** Sistema optimizado para carga perezosa (`lazy loading`) y manejo de errores (imágenes por defecto en caso de falla o ausencia).
- **Integración con WhatsApp:** Botón flotante para la captura y redirección de pedidos directamente al canal de atención del restaurante.
- **Formateo de Precios:** Custom pipes en Angular para el formateo automático de moneda local (COP).

## 🚀 Instalación y Ejecución Local

Sigue estos pasos para clonar y ejecutar el proyecto en tu entorno local:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/fast-food-menu.git
   cd fast-food-menu
   ```

2. **Instalar las dependencias:**
   ```bash
   npm install
   ```

3. **Levantar el servidor de desarrollo:**
   ```bash
   ng serve --open
   ```
   *La aplicación se abrirá automáticamente en tu navegador por defecto (usualmente en `http://localhost:4200/`).*
